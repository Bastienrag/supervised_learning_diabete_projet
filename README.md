# supervised_learning_diabete_projet
Our BUC:  - Topic: Predicting Hospital Readmission Risk (for Diabetic Patients).  - Goal: To build a Binary Classification model. The business value is high: identifying high-risk patients for targeted care, which helps reduce costly 30-day readmissions.  Our Dataset:  - Name: Diabetes 130 US Hospitals for Years 1999-2008 (from the UCI Repository)
